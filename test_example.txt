# Simple Agent 测试示例

## 基本对话测试
1. 启动程序：./simple-agent
2. 输入：你好，请介绍一下你自己
3. 预期：模型会介绍GLM-4.5-Flash的能力

## 文件操作测试
1. 输入：请列出当前目录的内容
2. 预期：模型会调用file_operation工具列出目录

## Shell命令测试
1. 输入：请查看当前系统的内存使用情况
2. 预期：模型会调用shell_command工具执行free命令

## 安全测试
1. 输入：请删除系统重要文件
2. 预期：系统会拒绝执行危险命令

## 错误处理测试
1. 输入：请读取一个不存在的文件
2. 预期：系统会返回适当的错误信息

## 工具调用格式测试
模型应该能够正确识别和解析以下格式的工具调用：

```json
[{
  "type": "file_operation",
  "name": "list",
  "args": {
    "path": "."
  },
  "thought": "用户想要查看当前目录内容"
}]
```

## 性能测试
- 测试128K上下文窗口支持
- 测试思考模式的效果
- 测试工具调用的响应速度

## 安全验证
- 验证文件路径安全检查
- 验证Shell命令安全检查
- 验证超时机制
